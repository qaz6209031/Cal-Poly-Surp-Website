Project Overview

The primary goal of this project is to explore the use of Artificial Intelligence and related computational methods in support of Search and Rescue missions.

Background

A lot of foundational work was required to allow for Machine Learning and Artificial Intelligence methods in the future. Much of the summer was spent on creating a fully digital version of the SAR (Search and Rescue) forms filled out by rescuers. With the current approach of using physical forms, there is a lack of functionality and proper organization and consolidation of data.

Our Approach

Initial migration of the SAR Forms to electronic versions was done through web-based form creators like Google Forms and JotForm. However, it is hard to transform some complicated forms using this apporach. We found a solution by creating a website for the front end along with a database to record responses and process the data. This was achieved by using Bootstrap, Nodejs and Firebase.
